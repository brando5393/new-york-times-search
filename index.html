About This Application

This application was constructed as part of an in class assignment for the University of Wisconsin Coding Bootcamp as part of the unit regarding APIs. Using this app, a search term can be given to return 1, 5 or 10 related articles from the New York Times. In addition to the search term, the app allows for an optional start and end year to be given to further filter results.
